# 📸 Photo Compressor

This is a simple **photo compression app** built with **Streamlit**. It allows you to compress images by:  
- Setting a target file size (in KB)  
- Adjusting the compression quality  

---

## 🚀 Features
✅ Supports **JPG, JPEG, and PNG** formats  
✅ Compress by quality or target size  
✅ Preview original and compressed images  
✅ Download the compressed image
